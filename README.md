# Product Catalog with Loaders
A product catalog app to learn React Router v7 loaders for data fetching.

## Getting Started
1. Clone: `git clone https://github.com/LoisBN/fpp-product-loader.git`
2. Install: `npm install`
3. Copy `.env.example` to `.env` and add your Supabase credentials
4. Run: `npm run dev`

## Your Tasks
- Create a `products` table in Supabase with sample data
- Implement a loader function to fetch products before the page renders
- Display a product grid using the loaded data
- Add a product detail page with its own loader
- Handle loading and error states
